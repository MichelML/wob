Generating white on black pictures, and more.  

### Install  
```  
git clone https://github.com/MichelML/wob.git \ 
cd wob \ 
sudo npm i -g .  
```  

### Usage  
```  
wob -m "Hello World"   
```  
This will output:   
<img src="helloworld.jpg" />  
Run the following to see all options:  
```  
wob -h   
```
